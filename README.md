# 폐선암 추적관찰 저염 레시피 50

정적 PWA 앱입니다. GitHub에 올리고 Vercel/Netlify/GitHub Pages로 배포하면 휴대폰에서 홈 화면에 추가해 앱처럼 사용할 수 있습니다.

## 포함 기능
- 폐선암 추적관찰 식단 기준의 저염 레시피 50개
- 검색, 분류 필터, 태그 필터
- 오늘 추천 3개
- 7일 식단 자동생성
- 장보기 메모 자동 생성
- 즐겨찾기 저장
- 오프라인 캐시

## 배포 방법
1. 이 폴더의 파일 전체를 GitHub 저장소에 업로드합니다.
2. Vercel에서 해당 저장소를 Import 합니다.
3. Framework Preset은 Other, Build Command는 비워두고 배포합니다.
4. 배포 URL을 iPhone/Android에서 열고 “홈 화면에 추가”를 누릅니다.

## 주의
이 앱은 치료나 재발 예방을 보장하지 않습니다. 주치의와 영양사의 지침이 우선입니다.
